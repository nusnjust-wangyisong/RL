# HER、课程学习、稳定性约束与精密伺服消融实验报告

本报告基于接触变力 `medium` 工况评估结果整理，用于说明完整方法各模块相对基础 TD3 的增益。

消融链路为：`TD3 -> TD3+HER -> TD3+HER+课程学习+稳定性约束（无伺服） -> 稳定性奖励强化（无伺服） -> 完整方法`。

其中“无伺服”表示评估端关闭近目标精密伺服，只观察训练阶段稳定性奖励、课程学习和动作平滑本身的效果。

完整方法对应 `TD3+HER+自适应课程学习+稳定性约束+多目标稳定精密伺服`。

## 关键提升

- 随机目标接触变力下，普通稳定性约束无伺服组的最终误差为 1.850 mm，5mm 成功率和 Hold@5mm 均为 1.000，说明训练阶段约束已经能保证 5mm 到达能力。
- 提高动作平滑、jerk 和振动惩罚权重后，稳定性奖励强化组在无伺服条件下将 RMS 从 7.164 降至 5.900，峰峰值从 44.416 降至 38.041，jerk RMS 从 162.208 降至 126.479。这说明稳定性奖励本身确实能压低受迫振动响应。
- 稳定性奖励强化组的 5mm 成功率为 0.967，略低于完整方法；完整方法通过多目标稳定精密伺服把成功率和 Hold@5mm 恢复到 1.000，并进一步把 RMS 降至 0.489、峰峰值降至 2.535、jerk RMS 降至 8.868。

## 结论

消融结果可以区分各模块作用：HER 与课程学习保证 5mm 到达能力和随机目标泛化；增强后的稳定性奖励能够在无伺服条件下降低 RMS、峰峰值和 jerk，证明稳定性约束不是无效项；多目标稳定精密伺服进一步解决“压振后精度略降”的问题，使最终方法同时达到 5mm 精度和低振动响应。因此，最终创新点应表述为训练阶段稳定性约束与执行阶段精密伺服的协同，而不是声称每个单独模块都独立最优。
